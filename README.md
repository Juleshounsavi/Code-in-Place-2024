# Code in Place 2024 — My First Steps in Python

This repository contains my work from **Code in Place 2024**, a free online 
program by Stanford University based on the first half of CS106A.

At the time, I was new to Python programming. These exercises from Karel 
the Robot to lists and dictionaries were one of my very first contact with 
programming logic in Python.

## What's inside

- **Diagnostics** — entry-level assessments (astronaut height, odd/even, Karel waves...)
- **Weekly assignments** — Weeks 1 to 6, covering Karel, variables, loops, 
functions, graphics, and data structures
